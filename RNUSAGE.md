# React Native 的使用



#### 创建一个RN工程

1.homebrew安装node.js

`brew install node`

2.安装Yarn、React Native的命令行工具（react-native-cli）

`npm install -g yarn react-native-cli`

3.安装推荐工具

`brew install watchman`

`brew install flow`
